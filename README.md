# 2dRandomTomo
Matlab code for the 2-dim random tomography paper

This is the Matlab for Two-Dimensional Tomography from Noisy Projections Taken at Unknown Random Directions published in SIAM Journal on Imaging SciencesVol. 6, Iss. 1 (2013)

DOI:10.1137/090764657

I didn't maintain this code since 2013, but it should be well functional.
